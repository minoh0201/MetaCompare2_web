Steps for using Metacompare :

  1. Login to the system using your credentials.
    If you do not have an account, use the guest account.
    Note : You cannot upload or run samples using guest account, but you can visualize the samples and see the result.
    Credentials for guest account:
       Username : guest
       Password : metacompare
    OR
    create a new account to access all the features.
  2. From navigation select ‘My Project’.
  3. Click on ‘Add Project’ [skip this step if you already have a project you want to work with].
  4. Type project name and click on ‘submit’ button[skip this step if you already have a project you want to work with].
  5. Click on ‘Upload Samples’ button.
  6. Read the given instructions and fill out the form.
    • Read the instructions on the website.
    • Select the project you want to add a new sample to.
    • Enter sample name.
    • Upload FASTA file containing assembled sequences.
    • Click on Add more samples [if required].
    • Click on ‘upload’ button.
  7. Click on ‘Run’ button in front of the sample you created in step 6.
  8. This will take some time depending upon the size of the sample FASTA file.
  9. You can see the results [resistome risk score and some other things] in front of each sample. The meaning of given results is explained here[some_url].
  10. To see visualizations of the scaffolds in terms of ARGs, MGEs and Pathogens, Click on ‘Visualize’ button.
  11. Click on ‘Visualize’ button in front of a scaffold to visualize that scaffold.
  12. Read the instructions and see the visualizations.
